# Changelog

All notable changes to Deriax will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Complete plugin system with dynamic loading
- Intelligent caching layer (memory, disk, Redis)
- Dynamic analysis with sandbox support
- Advanced static analysis (CFG, DFA, symbolic execution)
- Code emulation using Unicorn engine
- Machine learning for malware detection
- Threat intelligence integration (VirusTotal, OTX, MISP)
- Multi-format support (PE, ELF, Mach-O, DEX, WASM, .NET, Java, Python)
- Terminal User Interface (TUI)
- Web API and dashboard
- Comprehensive reporting (JSON, HTML, PDF, Markdown)
- YARA rules engine
- ROP gadget finder
- Vulnerability scanner
- CTF tools (crypto, pwn, forensics)

### Changed
- Improved performance with parallel processing
- Enhanced error handling
- Better documentation

### Fixed
- Various bug fixes

## [0.1.0] - 2025-12-05

### Added
- Initial project structure
- Basic PE and ELF parsing
- Simple disassembly
- Malware signature detection
- Basic CLI interface
